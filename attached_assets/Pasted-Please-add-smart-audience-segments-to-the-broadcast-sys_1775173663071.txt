═══════════════════════════════
SMART SEGMENTS
═══════════════════════════════

Add these options to the recipient selector dropdown:

1. "Inactive users (30+ days)"
   - Users who have not logged in for 30 or more days
   - Query: users where last_active < NOW() - INTERVAL '30 days'
   - Show count dynamically

2. "Inactive users (14+ days)"
   - Users who have not logged in for 14 or more days
   - Query: users where last_active < NOW() - INTERVAL '14 days'

3. "Incomplete profiles"
   - Users missing any of: profile photo, bio, or country of origin
   - Query: users where photo_url IS NULL OR bio IS NULL OR heritage IS NULL
   - Show count dynamically

4. "No matches yet"
   - Registered users who have zero mutual matches
   - Query: users not present in matches/conversations table
   - Show count dynamically

5. "Free tier — never upgraded"
   - Users who have been free tier for 14+ days
   - Good for upgrade nudge campaigns
   - Query: users where tier = 'free' AND created_at < NOW() - INTERVAL '14 days'

6. "New this week"
   - Users who registered in the last 7 days
   - Good for welcome and onboarding emails
   - Query: users where created_at > NOW() - INTERVAL '7 days'

7. "Core members"
   - Already exists ✅

8. "Serious Badge members"
   - Already exists ✅

9. "Waitlist only (not yet registered)"
   - Emails in early_access table that do NOT have a matching account in users table
   - These are people who signed up for early access but never created an account

LAST ACTIVE TRACKING

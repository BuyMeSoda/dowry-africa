Please implement the free tier daily limits with an "upgrade interest" capture instead of a payment button, since payment integration is not yet live.

═══════════════════════════════
1. FREE TIER DAILY LIMITS
═══════════════════════════════
Free tier users can send messages and likes with daily limits:
- Messages per day: 3
- Likes per day: 10
- Limits reset at midnight UTC

Paid tiers (Core and Serious Badge):
- Unlimited messages
- Unlimited likes

═══════════════════════════════
2. DATABASE
═══════════════════════════════
Add user_daily_limits table:
- id
- user_id (foreign key)
- date (date)
- messages_sent (integer, default 0)
- likes_sent (integer, default 0)
- UNIQUE constraint on (user_id, date)

Add upgrade_interest table:
- id
- user_id (foreign key, nullable)
- email (text)
- plan_interest (text: 'core' or 'badge')
- created_at
- UNIQUE on (email, plan_interest)

Add to settings table:
- free_daily_message_limit (integer, default 3)
- free_daily_like_limit (integer, default 10)
- payments_live (boolean, default false)

═══════════════════════════════
3. BACKEND
═══════════════════════════════
Update POST /api/messages/:userId:
- If free tier:
  * Check today's message count
  * If >= limit: return 429 { error: 'daily_limit_reached', messagesUsed, messagesLimit, remaining: 0 }
  * If under limit: send message, increment counter
- If paid tier: no limit

Update POST /api/likes/:userId:
- If free tier:
  * Check today's like count
  * If >= limit: return 429 { error: 'daily_limit_reached' }
  * If under limit: allow like, increment counter
- If paid tier: no limit

Add POST /api/payments/upgrade-interest:
- Public endpoint (auth required)
- Body: { plan: 'core' | 'badge' }
- Save to upgrade_interest table with user email
- Return success
- If already registered: return { alreadyRegistered: true }

Add GET /api/payments/status update:
- Include daily limits info:
{
  tier,
  paymentsLive: false,
  dailyLimits: {
    messagesLimit: 3,
    messagesUsed: 1,
    messagesRemaining: 2,
    likesLimit: 10,
    likesUsed: 0,
    likesRemaining: 10,
    resetsAt: midnight UTC timestamp
  }
}

Please add email templates to the Communications broadcast section in the admin panel.

1. TEMPLATE SELECTOR
Above the Subject and Body fields in the Broadcast compose form, add a "Use a template" dropdown with these pre-built templates.

═══════════════════════════════
EMAIL DESIGN REQUIREMENTS
═══════════════════════════════
All email templates must look premium and on-brand — not like a plain text email. Use this HTML structure for every template:

- Max width: 600px, centered
- Background: #f9f6f2 (warm off-white, matches the app)
- Email card: white background, border-radius 12px, subtle shadow
- Header: rose/burgundy background (#8B1C3F) with Dowry.Africa logo and name in white
- Body: clean serif font for headings, clean sans-serif for body text
- CTA Button: rose/burgundy background (#8B1C3F), white text, border-radius 50px, padding 14px 32px
- Divider lines: light gray
- Footer: dark gray background, white text, small font
- Mobile responsive — single column layout

IMPORTANT: All URLs in templates must use process.env.FRONTEND_URL as the base URL, not any hardcoded domain. Example: ${process.env.FRONTEND_URL}/register — this way when the custom domain is set up, only the FRONTEND_URL environment variable needs updating in Railway and all email links update automatically.

5. EMAIL PREVIEW
═══════════════════════════════
In the admin broadcast compose form, add a "Preview email" button that opens a modal showing exactly how the rendered HTML email will look — with the rose/burgundy header, formatted body, CTA button, and footer. This way admin can see the final design before sending.

═══════════════════════════════
6. ENVIRONMENT VARIABLES
═══════════════════════════════
Make sure these are used throughout:
- RESEND_API_KEY — for sending emails
- FROM_EMAIL — sender address (hello@dowry.africa)
- FRONTEND_URL — base URL for all links in emails

When custom domain is ready, only FRONTEND_URL needs updating in Railway:
FRONTEND_URL = https://dowry.africa
All email links will automatically use the new domain.

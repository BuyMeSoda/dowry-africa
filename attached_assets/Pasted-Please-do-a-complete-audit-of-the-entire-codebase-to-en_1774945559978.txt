1. HARDCODED VALUES AUDIT
═══════════════════════════════
Search the entire codebase for hardcoded values that should be environment variables or dynamic:

- Any hardcoded URLs (localhost, railway URLs, specific domains)
- Any hardcoded API keys or secrets
- Any hardcoded prices ($7, $15, $12.99, $19.99) that should come from the database/API
- Any hardcoded user IDs or demo data references in production code
- Any hardcoded email addresses
- Any hardcoded Stripe price IDs or product IDs
- Any hardcoded Cloudinary URLs or cloud names

For each hardcoded value found:
- If it should be an environment variable: replace with process.env.VARIABLE_NAME (backend) or import.meta.env.VITE_VARIABLE_NAME (frontend)
- If it should come from the database/API: replace with a dynamic fetch
- If it is legitimate (e.g. a UI label): leave it but note it in a comment
